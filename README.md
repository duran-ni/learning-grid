# Grid- Paleta de Colores

# 📝Descripción
El objetivo del proyecto es reproducir una página web que muestra una paleta de colores de fondo CSS nombrados. La página presenta 15 colores (black, teal, blue, navy, lime, white, purple, yellow, olive, red, maroon, gray, olivedrab, deep sky blue, fuchsia y green) organizados en tarjetas, cada una con su color de fondo correspondiente y el nombre del color como texto.
La página incluye un encabezado con título y subtítulo, una barra de cabecera negra con el texto "Colors" y la cuadrícula de tarjetas de colores. El diseño debe ser totalmente responsive, adaptándose a: Phone, Tablet y Desktop.

# 🔍Análisis
La página muestra una paleta de colores CSS organizados en tarjetas dentro de una cuadrícula. Cada tarjeta tiene el color como fondo y su nombre como texto.
El layout cambia según el dispositivo: en phone la grid tiene 3 columnas, en tablet 4 y en desktop 6. En todos los breakpoints, la barra negra "Colors" siempre ocupa el ancho completo, mientras que la tarjeta "green" en desktop pasa a abarcar 3 columnas en lugar de una.
Respecto al texto, la mayoría de tarjetas usan texto blanco, salvo white, lime y yellow, que necesitan texto oscuro para garantizar contraste suficiente. La tarjeta white además requiere un borde que la diferencie del fondo gris de la página.
